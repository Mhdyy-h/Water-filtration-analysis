# 🚰 Analyse de Données : Système de Filtration d’Eau Intelligent

## 📝 Présentation du Projet
Ce projet consiste en une analyse statistique complète d'un jeu de données simulant l'acceptation d'un système de filtration d'eau écologique connecté à une IA. L'objectif est de comprendre les facteurs qui influencent l'intention d'achat et le prix accepté par les consommateurs.

## 🎯 Objectifs
- Analyser le profil socio-économique des utilisateurs potentiels.
- Étudier la corrélation entre la confiance en l'IA et la sensibilité au développement durable.
- Identifier les segments de clients prêts à investir dans cette technologie.

## 🛠️ Technologies Utilisées
- **Langage :** Python 🐍
- **Librairies :** Pandas, Matplotlib, Seaborn.
- **Outil :** Jupyter Notebook.

## 📂 Contenu du Dépôt
- `filtration_deau.ipynb` : Le notebook contenant tout le code et les visualisations.
- `dataset_filtration_eau.csv` : Le jeu de données utilisé pour l'analyse.

## 🚀 Comment exécuter ce projet ?
1. Cloner le dépôt : `git clone https://github.com/Mhdyy-h/Water-filtration-analysis.git`
2. Installer les dépendances : `pip install pandas matplotlib seaborn`
3. Ouvrir le fichier `.ipynb` avec Jupyter ou VS Code.
